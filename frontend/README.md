# Makereal Labs cafe music system frontend

A simple system for playing music in club, this is the frontend code

## Install dependencies

Run `npm i` to fetch the required packages

## Build & Run

Launch development server: `npm run dev`

Build: `npm run build`

## Configure

Edit `SERVER_URL` in [`src/session.ts`](src/session.ts) to point to the backend port

If you're doing local development, configuring `SERVER_URL` to `"ws://127.0.0.1:9001"` should suffice
